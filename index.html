Proiectarea interfetelor grafice pentru monitorizare si control

Ma numesc Calburean Ruxandra si studiez specializarea Sisteme de Monitorizare si Control in Inginerie Electrica in cadrul Facultatii de Inginerie Electrica, Universitatea Tehnica din Cluj Napoca. Prezentul proiect trateaza dezvoltarea unei aplicatii web pentru conversia valorilor electrice, precum si un quizz pentru evaluarea cunostiintelor generale". Limbajul utilizat pentru realizarea structurii aplicatiei web este HTML, iar pentru stilizarea acesteia s-a utilizat limbajul CSS, integrat in codul HTML. Pentru realizarea functionalitatilor de conversie a valorilor electrice si a quizz-ului s-a utilizat limbajul de programare JavaScript.
